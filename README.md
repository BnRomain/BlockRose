# Block Rose – Front-end website

## Présentation du projet
Conception et développement du front-end d’un site web de location de matériel de production, réalisé à partir d’un brand book fourni.

## Contexte et cahier des charges
- Respect strict de l’identité visuelle (typographies, couleurs, iconographie)
- Création d’une interface claire et cohérente
- Développement en HTML, CSS et JavaScript vanilla

## Ce que j’ai réalisé
- Intégration fidèle au brand book
- Structuration HTML sémantique
- Mise en page responsive
- Interactions simples en JavaScript

## Difficultés rencontrées
- Traduire un brand book en composants concrets
- Trouver un équilibre entre créativité et contraintes graphiques
- Organisation du code sur un premier projet front-end

## Ce que le projet m’a appris
- Travailler à partir d’un cahier des charges réel
- Respecter une identité visuelle existante
- Approche méthodique du développement front-end
